# EpisodeX

## Overview
EpisodeX is an awesome app to help you keep track of your progress with your favourite Tv Shows.

## Installation
Run the following commands:

```bash
bundle install
rake db:migrate
rackup
```

## Testing
Run the following command:

```bash
rake test
```
